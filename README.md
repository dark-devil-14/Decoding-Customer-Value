# Decoding Customer Value: Strategic Intelligence & Retention Analysis

## 📌 Project Overview
This project transforms raw transactional data into actionable customer intelligence. By leveraging Python for feature engineering, SQL for deep segmentation, and Power BI for executive reporting, I have developed a **Margin-First Retention Strategy** designed to transition the brand from discount-led growth to long-term profitability.

## 📂 Repository Structure

### 1. [reports/](./reports/)
- **[Executive_Summary.md](./reports/Executive_Summary.md)**: A high-level brief for stakeholders outlining key findings and the implementation timeline.
- **[Retention_Playbook.md](./reports/Retention_Playbook.md)**: The strategic "Promotional Sunset Plan" and Ideal Customer Profile (ICP) definitions.
- **[Technical_Documentation.md](./reports/Technical_Documentation.md)**: Detailed documentation of the preprocessing journey, feature engineering (eCLV, Engagement Score), and statistical validation.

### 2. [sql-based-customer-segmentation-analysis/](./sql-based-customer-segmentation-analysis/)
Contains the SQL logic layer used to answer core business questions:
- `loyal_customer.sql`: Identifying "Genuinely Loyal" vs. "Bargain Hunters."
- `ideal_customer_profile.sql`: Defining the characteristics of the top 10% of customers.
- `geographic_credibility.sql`: Regional analysis of organic demand.

### 3. [data/](./data/)
- `Dataset.csv`: Raw transactional source.
- `Gold_Customer_Data.csv`: The final, feature-rich "Gold" dataset ready for modeling.

### 4. [tables/](./tables/)
- Exported query results used for visualization and reporting.

## 🚀 Key Achievements
- **Engineered behavioral metrics** including a multi-factor Engagement Score and estimated CLV.
- **Statistically validated segments** using t-tests and ANOVA to ensure high-confidence targeting.
- **Developed a Strategic Roadmap** to protect margins by phasing out high-risk discounting.
